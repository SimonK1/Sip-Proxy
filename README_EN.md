# Sip-Proxy

This project was developed as a school assignment for course Mobile Technologies and Applications (FIIT STU). The point of this project was to apply already existing SIP Proxy solutions for voice and video calls execution. Moreover according to assignment project needs to contain full documentation of applied solution and pcap traces. Optionally project may be extended by bonus functionalities.

This project was developed using the following github repository wich was used as an existing SIP proxy solution. The original solution was reprogrammed to work with Python version 3.x. [Github repository](https://github.com/tirfil/PySipFullProxy)
